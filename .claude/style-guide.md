# Writing Style Guide

This file defines the voice and tone for all content on Agentic Playbook. Every agent writing or editing content must follow these rules.

## Voice

Write like a senior engineer explaining something to a peer. Short sentences. Direct. No filler.

## Banned Phrases

Never use these in any page, description, or commit message:

- "No marketing fluff"
- "No theory"
- "Real commands, real configs, real troubleshooting"
- "Battle-tested"
- "Production-grade"
- "Hands-on"
- "Opinionated"
- "The tools shaping the agent era"
- "And beyond"
- "Just works"
- "Cutting-edge"
- "Game-changer"
- "Supercharge"
- "Unlock the power of"
- "Dive deep into"
- "Comprehensive guide to"
- "Everything you need to know"
- "In this guide, we will..."
- "Let's get started!"
- "Without further ado"

These are AI writing crutches. They add no information and make the site sound generated.

## What to write instead

Bad: "A hands-on, production-grade, battle-tested guide to deploying agents"
Good: "How to deploy an OpenClaw agent to production"

Bad: "No marketing fluff — just real commands and real configs"
Good: "Agentic Playbook is a collection of tested guides for setting up and running AI agent systems in production."

Bad: "Dive deep into the architecture of multi-agent systems"
Good: "How the multi-agent gateway works"

## Structure rules

- Headings should describe what the section contains, not sell it
- First paragraph of any page: one sentence saying what this page helps you do
- Use bullet lists for 3+ items. Use plain text for 1-2 items.
- Tables: only use when comparing structured data with 3+ columns. For simple lists (name + description), use a bullet list instead
- Code blocks: always include the language tag
- Links: use descriptive text, not "click here"

## Tone calibration

- Second person ("you") when addressing the reader
- Present tense for instructions ("Run this command" not "You will run this command")
- Don't explain why documentation is good. Just write good documentation.
- Don't announce what you're about to say. Just say it.
