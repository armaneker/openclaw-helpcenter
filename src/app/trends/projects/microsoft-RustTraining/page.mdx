# Microsoft Rust Training

A comprehensive collection of seven training courses covering Rust from beginner to expert level. The material is organized by programming background and complexity, offering structured learning paths for different experience levels.

## Key Features

- **Seven specialized books** covering different programming backgrounds (C/C++, C#, Python)
- **Difficulty-graded content** from bridge courses to expert-level patterns
- **Deep-dive modules** on async programming, advanced patterns, and engineering practices
- **Interactive learning** with Rust playgrounds, Mermaid diagrams, and exercises
- **Full-text search** and keyboard navigation in rendered format
- **Production-ready content** covering tooling, CI/CD, and cross-compilation

## Installation

Clone the repository and install the required tools:

```bash
git clone https://github.com/microsoft/RustTraining.git
cd RustTraining

# Install mdBook and mermaid plugin
cargo install mdbook@0.4.52 mdbook-mermaid@0.14.0

# Serve all books locally
cargo xtask serve
```

## Basic Usage

```bash
# Build all books
cargo xtask build

# Serve locally at http://localhost:3000
cargo xtask serve

# Build individual book
cd python-book && mdbook serve --open

# Clean build artifacts
cargo xtask clean
```

<Callout type="tip">
Books are organized by experience level: Bridge (🟢), Deep Dive (🔵), Advanced (🟡), Expert (🟣), and Practices (🟤). Start with the bridge book matching your background.
</Callout>
