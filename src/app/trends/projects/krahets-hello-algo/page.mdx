# Hello Algo

Hello Algo is an open-source interactive tutorial that teaches data structures and algorithms through animations and runnable code examples. The book supports multiple languages and provides implementations in 13+ programming languages.

## Key Features

- **Animated explanations** - Visual animations illustrate how algorithms work step-by-step
- **Interactive code** - Run algorithm implementations directly in your browser
- **Multi-language support** - Available in Chinese (Simplified/Traditional), English, Japanese, and Russian
- **Comprehensive implementations** - Code examples in Python, Java, C++, C, C#, JavaScript, Go, Swift, Rust, Ruby, Kotlin, TypeScript, and Dart
- **Beginner-friendly** - Designed with a smooth learning curve for algorithm newcomers
- **Community-driven** - Active contribution and discussion from learners

## Installation

Access the tutorial online or download offline versions:

```bash
# Read online
https://www.hello-algo.com/

# Download PDF/EPUB from releases
https://github.com/krahets/hello-algo/releases
```

For local development:

```bash
git clone https://github.com/krahets/hello-algo.git
cd hello-algo
```

## Basic Usage

Navigate to any algorithm topic on the website and:

1. Read the animated explanation
2. Run code examples in your preferred language
3. Experiment with different inputs
4. Participate in community discussions

<Callout type="tip">
The repository includes runnable code for each algorithm across 13+ programming languages, making it easy to see implementations in your preferred syntax.
</Callout>
