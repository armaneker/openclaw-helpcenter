# Paperclip

Paperclip is an open-source Node.js server and React UI that orchestrates teams of AI agents to run autonomous businesses. It provides organizational structure, budgets, governance, and coordination for multi-agent systems.

## Key Features

- **Bring Your Own Agent** — Integrates any agent (OpenClaw, Claude, Codex, Cursor) via heartbeat protocol
- **Goal Alignment** — Every task traces back to company mission with clear context flow
- **Cost Control** — Monthly budgets per agent with automatic throttling to prevent runaway costs
- **Org Charts** — Hierarchical structure with roles, reporting lines, and delegation
- **Ticket System** — Immutable audit log with full tool-call tracing and conversation threading
- **Multi-Company** — Run multiple autonomous businesses from one deployment with data isolation
- **Mobile Ready** — Monitor and manage operations from anywhere

## Installation

```bash
git clone https://github.com/paperclipai/paperclip.git
cd paperclip
npm install
npm run dev
```

<Callout type="info">
Paperclip requires Node.js and supports any agent that can receive heartbeat signals.
</Callout>
