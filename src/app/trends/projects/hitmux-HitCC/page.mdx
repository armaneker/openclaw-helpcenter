# HitCC

HitCC is a comprehensive reverse-engineering documentation project that analyzes the complete runtime logic of Claude Code CLI Node.js version v2.1.84. It provides structured documentation for understanding the system architecture, tool execution, and surrounding ecosystem without containing any original source code.

## Key Features

- Complete documentation of Claude Code CLI's execution chain from startup to agent loop
- Analysis of core systems: MCP, Plugin, Skill, TUI, Remote Persistence, and Bridge components  
- Topic-oriented breakdown covering tool use, prompt assembly, session persistence, and permission systems
- Rewrite-oriented architectural guidance with candidate structures and open questions
- 81 documentation files covering 27,170 lines of analysis across runtime, execution, and ecosystem topics

## Installation

The project provides documentation only. To analyze the original package:

```sh
npm pack @anthropic-ai/claude-code@2.1.84
```

Use the Python recovery tools for initial cleanup of obfuscated code:

```python
# Python scripts available under recovery_tools/
python recovery_tools/cleanup.py
```

## Basic Usage

<Callout type="info">
This is documentation, not executable code. Use it as reference for understanding or rewriting similar systems.
