# AI Engineering from Scratch

A comprehensive learning course with 260+ lessons across 20 phases covering everything from linear algebra to autonomous agent swarms. Each lesson produces reusable artifacts: prompts, skills, agents, and MCP servers.

## Key Features

- **Complete AI pipeline**: Math foundations → ML → Deep Learning → NLP → Computer Vision → LLMs → Agent swarms
- **Multi-language approach**: Python, TypeScript, Rust, and Julia implementations
- **Build-first methodology**: Implement algorithms from scratch before using frameworks
- **Practical outputs**: Every lesson creates tools, agents, or skills you can actually use
- **Progressive complexity**: 20 structured phases from setup to advanced swarm intelligence
- **Real implementations**: Runnable code, notebooks, documentation, and web applications

## Installation

Clone the repository and set up your environment:

```bash
git clone https://github.com/rohitg00/ai-engineering-from-scratch.git
cd ai-engineering-from-scratch

# Follow Phase 0 lessons for complete environment setup
# Each phase has specific requirements detailed in its README
```

<Callout type="info">
Start with Phase 0 (Setup & Tooling) which covers dev environment, GPU setup, Docker configuration, and dependency management across all supported languages.
</Callout>

## Basic Usage

Navigate through phases sequentially. Each lesson includes theory, implementation, and practical exercises:

```bash
# Start with Phase 0
cd phases/00-setup-and-tooling/01-dev-environment/

# Each lesson contains:
# - README.md with explanations
# - Code implementations
# - Jupyter notebooks where applicable
# - Tests and examples
```

<Callout type="tip">
The course is designed to be language-agnostic where possible. Choose your preferred language for each lesson, though some concepts work better in specific languages (Julia for math, TypeScript for web agents).
</Callout>
