# MiniMax Skills

Development skills for AI coding agents that plug into your favorite AI coding tool. Provides structured, production-quality guidance for frontend, fullstack, Android, iOS, and shader development.

## What it does

- **Frontend Development** — React/Next.js with Tailwind CSS, Framer Motion animations, and MiniMax API integrations
- **Fullstack Architecture** — REST APIs, auth flows, real-time features, database integration
- **Mobile Development** — Android (Kotlin/Jetpack Compose) and iOS (UIKit/SwiftUI) applications
- **Shader Programming** — GLSL techniques for visual effects, ray marching, SDF modeling
- **Document Generation** — PDF, PPTX, DOCX, and Excel file creation and manipulation
- **Media Processing** — Animated GIF sticker generation from photos

## Installation

### Claude Code

```bash
claude plugin marketplace add https://github.com/MiniMax-AI/skills
claude plugin install minimax-skills
```

### Cursor

```bash
git clone https://github.com/MiniMax-AI/skills.git ~/.cursor/minimax-skills
```

Add the skills path `~/.cursor/minimax-skills/skills/` to your Cursor settings.

### Other Editors

```bash
git clone https://github.com/MiniMax-AI/skills.git ~/.minimax-skills
```

Create symlinks to the skills directory based on your editor's configuration requirements.

## Basic Usage

Once installed, reference skills in your AI coding sessions:

```markdown
Use the frontend-dev skill to build a React component with Framer Motion animations
```

```markdown
Apply the fullstack-dev skill to design a REST API with JWT authentication
```

Each skill provides structured guidance following best practices for the specific development domain.
