# Freqtrade Bot Configuration Tool

A Python-based configuration and management tool for the Freqtrade open-source crypto trading bot. It provides a Rich terminal interface with interactive menus for strategy backtesting, live/dry-run trading, and exchange management.

## Key Features

- Strategy backtesting and hyperparameter optimization with FreqAI machine learning support
- Live and dry-run trading modes across 10+ exchanges (Binance, Bybit, OKX, Kraken, Gate.io)
- FreqUI web dashboard and Telegram bot integration
- Rich terminal menu interface with interactive navigation
- Automated virtual environment setup and dependency management
- Cross-platform support with Windows one-click launcher

## Installation

<Callout type="info">
Requires Python 3.11+ and the core Freqtrade package installed separately.
</Callout>

```bash
git clone https://github.com/Kaleighc793/freqtrade-bot
cd freqtrade-bot
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt

# Install core Freqtrade
pip install freqtrade
```

For Windows users, simply double-click `run.cmd` for automatic setup.

## Basic Usage

Launch the interactive menu:

```bash
python main.py
```

Create initial configuration:

```bash
# Through menu option 4, or directly:
freqtrade create-userdir --userdir user_data
freqtrade new-config --config user_data/config.json
```

<Callout type="tip">
Start with dry-run mode (`"dry_run": true`) to test strategies without real money.
</Callout>

Example backtest command generated by the tool:

```bash
freqtrade backtesting --strategy SampleStrategy --timerange 20230101-20231201
```
