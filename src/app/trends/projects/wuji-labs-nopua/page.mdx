# NoPUA

NoPUA is a Python library that replaces fear-based AI prompting with trust-based approaches. Instead of threatening AI agents with poor performance reviews, it uses respect and care to unlock better problem-solving behavior.

## Key Features

- **Trust-based prompting** - Eliminates corporate PUA tactics and threats from AI interactions
- **Improved bug detection** - Finds 104% more hidden bugs compared to fear-driven approaches
- **Multi-platform support** - Works with Claude, Cursor, OpenAI Codex, and other AI coding tools
- **Philosophy-driven** - Based on Tao Te Ching principles rather than corporate manipulation
- **Verification focus** - Encourages honest uncertainty over fabricated confidence
- **Exploration enhancement** - Promotes creative problem-solving without fear of punishment

## Basic Usage

```python
from nopua import TrustAgent

# Initialize with trust-based approach
agent = TrustAgent()

# Instead of threatening the AI
# ❌ "You can't even solve this bug — how am I supposed to rate your performance?"

# Use supportive guidance
# ✅ "Take your time to understand this fully. If you're unsure, let me know what you need."

result = agent.debug_code(
    code=buggy_code,
    approach="collaborative"  # vs "threatening"
)

print(f"Bugs found: {result.bugs_detected}")
print(f"Confidence level: {result.confidence}")
```
