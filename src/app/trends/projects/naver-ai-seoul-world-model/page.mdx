# Seoul World Model

Seoul World Model is a research project that grounds world simulation models using real-world data from Seoul metropolis. It generates realistic street-view sequences and urban environments based on actual metropolitan infrastructure and dynamics.

## Key Features

- Real-world grounded simulation using Seoul metropolitan data
- Street-view interpolation for generating realistic urban sequences
- Synthetic training data generation for world modeling
- Model checkpoints for inference and evaluation
- Research-grade implementation for urban simulation tasks
