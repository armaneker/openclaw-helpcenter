# free-for.dev

A curated list of SaaS, PaaS, and IaaS services that offer free tiers for developers and infrastructure engineers. This community-maintained resource helps you discover free services across major cloud providers, monitoring tools, CI/CD platforms, and more.

## Key Features

- **Comprehensive coverage** — Over 50 categories including cloud providers, monitoring, CI/CD, databases, and security tools
- **Infrastructure focus** — Services relevant to DevOps practitioners and system administrators
- **Quality criteria** — Only includes services with genuine free tiers (not trials), lasting at least one year
- **Community-driven** — Maintained by 1600+ contributors with active pull request reviews
- **Always up-to-date** — Regular updates to remove discontinued services and add new offerings

## Installation

This is a reference list hosted on GitHub. No installation required.

```bash
# Clone the repository for offline access
git clone https://github.com/ripienaar/free-for-dev.git

# Or bookmark the GitHub Pages site
open https://free-for.dev
```

## Basic Usage

Browse the categorized list to find free services for your infrastructure needs:

```markdown
# Example categories:
- Major Cloud Providers (AWS, GCP, Azure free tiers)
- Monitoring (Application and infrastructure monitoring)
- CI/CD (Build and deployment pipelines)
- DNS (Domain name services)
- CDN and Protection (Content delivery networks)
- Databases (Managed database services)
```

<Callout type="info">
Services must offer permanent free tiers, not just trials. Security-focused: TLS restrictions disqualify services from inclusion.
</Callout>
