# The Minimalist Entrepreneur Claude Skills

A collection of 10 Claude Code skills that implement business principles from Sahil Lavingia's "The Minimalist Entrepreneur" book. These skills guide you through the entire entrepreneurial journey from finding your community to scaling sustainably.

## Key Features

- **Complete business framework** — 10 skills covering community finding through sustainable growth
- **Sequential progression** — Skills follow the book's logical business development stages
- **Practical commands** — Each skill provides actionable guidance for specific business decisions
- **Manual-first approach** — Emphasizes procesizing before building automated solutions
- **Sustainable focus** — Built around profitable, bootstrapped business principles

## Installation

Install directly from the Claude Code marketplace:

```bash
/plugin marketplace add slavingia/skills
/plugin install minimalist-entrepreneur
```

Alternatively, install from a local clone:

```bash
git clone https://github.com/slavingia/skills.git ~/.claude/plugins/skills
```

Then in Claude Code:

```bash
/plugin marketplace add ~/.claude/plugins/skills
/plugin install minimalist-entrepreneur
```

## Basic Usage

Each skill maps to a specific business stage:

```bash
# Finding your initial business community
/find-community

# Testing business ideas before building
/validate-idea

# Building your minimum viable product
/mvp

# Converting ideas to manual processes
/processize

# Acquiring your first 100 customers
/first-customers

# Setting sustainable pricing strategy
/pricing
```

<Callout type="tip">
Use `/minimalist-review` to apply minimalist principles to any business decision throughout your journey.
</Callout>
