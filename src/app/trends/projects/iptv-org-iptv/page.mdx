# IPTV Channels Collection

A community-maintained repository of publicly available IPTV channels from around the world, organized into M3U playlists. The project aggregates streaming links for Internet Protocol television channels and provides structured access through various playlist formats.

## Key Features

- **Global channel collection** - Thousands of IPTV channels from all regions
- **M3U playlist format** - Compatible with most video players and IPTV apps
- **Categorized playlists** - Organized by country, language, and content type
- **EPG support** - Electronic Program Guide data available for most channels
- **API access** - Programmatic access to channel database
- **Regular updates** - Automated workflows maintain current channel availability

## Installation

No installation required. Access playlists directly via URL or clone the repository:

```bash
git clone https://github.com/iptv-org/iptv.git
```

## Basic Usage

Load the main playlist containing all channels:

```
https://iptv-org.github.io/iptv/index.m3u
```

Copy this URL into any IPTV player (VLC, Kodi, etc.) or use specific regional playlists from the PLAYLISTS.md file.

<Callout type="info">
Channel data comes from the separate iptv-org/database repository. Report channel issues there, not in the main repo.
</Callout>

<Callout type="warning">
This repository only contains links to publicly available streams. No video content is hosted directly.
</Callout>
