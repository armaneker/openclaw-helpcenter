# HelloGitHub

HelloGitHub is a monthly publication that curates interesting, beginner-friendly open source projects from GitHub. It publishes on the 28th of each month, featuring projects across multiple categories to help developers discover the joy of open source.

## Key Features

- **Monthly curation** of entry-level open source projects
- **Multi-category coverage** including interesting projects, open source books, practical projects, and enterprise-level solutions
- **Beginner-focused** selection criteria for accessibility
- **Multiple formats** available through website, WeChat, and GitHub
- **Community contributions** with project recommendations and self-nominations
- **Multilingual support** with Chinese, English, and Japanese documentation

## Installation

Clone the repository to access the monthly issues:

```bash
git clone https://github.com/521xueweihan/HelloGitHub.git
cd HelloGitHub
```

## Usage

Browse monthly issues in the `/content` directory:

```bash
# View the latest issue
cat content/HelloGitHub120.md

# Browse all available issues
ls content/HelloGitHub*.md
```

Access content through multiple channels:
- **Website**: [hellogithub.com](https://hellogithub.com/)
- **GitHub**: Monthly markdown files in the content directory
- **WeChat**: Official WeChat public account

<Callout type="info">
New issues are published monthly on the 28th. Each issue contains carefully selected projects with descriptions, screenshots, and links to help you quickly understand what makes each project interesting.
</Callout>
