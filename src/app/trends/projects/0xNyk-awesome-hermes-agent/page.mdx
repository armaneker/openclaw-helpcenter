# Awesome Hermes Agent

A curated list of skills, tools, integrations, and resources for Hermes Agent by Nous Research. This is the ecosystem tracker for the self-improving AI agent with built-in learning loops.

## What it does

- **Skill collections** — Community-built skills for goal management, literate programming, incident response, and personal productivity
- **Deployment tools** — Utilities for running Hermes on VPS, clusters, or serverless infrastructure  
- **Integrations** — Bridges to other agent frameworks, messaging platforms, and development tools
- **GUI frontends** — Web interfaces like hermes-workspace and mission-control for visual agent management
- **Resource discovery** — Organized catalog with maturity tags (production, beta, experimental) to guide selection

<Callout type="info">
The repository includes a three-step onboarding path: get Hermes running, add your first skills, then set up a GUI workspace.
</Callout>

## Basic usage

Navigate the ecosystem by maturity level and use case. Start with production-tagged resources for stability:

```markdown
| Tag | Meaning |
|-----|---------|
| production | Stable, documented, actively maintained |
| beta | Works but evolving — expect rough edges |
| experimental | Proof of concept — learn from it, don't depend on it |
```

For GUI management, the repository recommends hermes-workspace (500+ stars) or mission-control (3.7k+ stars) for fleet orchestration.

<Callout type="tip">
The main Hermes Agent repository has 23k+ stars and includes automatic migration from OpenClaw if you're switching agents.
</Callout>
