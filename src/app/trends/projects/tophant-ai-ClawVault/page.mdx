# ClawVault

ClawVault is a security vault for AI agents that provides atomic control over every AI interaction. It acts as a transparent proxy gateway to monitor, guard, and audit AI agent communications with external APIs.

## Key Features

- **Sensitive Data Detection** — Detects API keys, passwords, PII, credit cards across 15+ pattern types
- **Prompt Injection Defense** — Blocks role hijacking, instruction override, and data exfiltration attempts
- **Dangerous Command Guard** — Intercepts risky commands like `rm -rf`, `curl|bash`, privilege escalation
- **Auto-Sanitization** — Replaces secrets with placeholders, restores on response
- **Token Budget Control** — Daily/monthly limits with cost alerts
- **Real-time Dashboard** — Web UI for per-agent configuration and detection monitoring

## Basic Usage

```bash
# Start the proxy gateway and dashboard
clawvault start

# Scan text for sensitive data
clawvault scan "password=MySecret key=sk-proj-abc123"

# Interactive demo
clawvault demo
```

<Callout type="info">
ClawVault runs a transparent proxy on port 8765 that intercepts traffic between AI tools and external APIs (OpenAI, Anthropic, etc.).
</Callout>

### Using as OpenClaw Skill

```bash
# Quick setup
/clawvault install --mode quick

# Generate security rules
/clawvault generate-rule "Block AWS credentials"

# Test detection capabilities
/clawvault test --category all
```

## Configuration

```yaml
# ~/.ClawVault/config.yaml
proxy:
  port: 8765
  intercept_hosts: ["api.openai.com", "api.anthropic.com"]

guard:
  mode: "interactive"  # interactive | strict | permissive

monitor:
  daily_token_budget: 50000
```
