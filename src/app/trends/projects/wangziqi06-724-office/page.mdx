# 724 Office

724 Office is a self-evolving AI agent system built in 3,500 lines of pure Python with zero framework dependencies. It runs 24/7 in production with 26 built-in tools, three-layer memory, and runtime tool creation capabilities.

## Key Features

- **Tool Use Loop** — OpenAI-compatible function calling with automatic retry, up to 20 iterations per conversation
- **Three-Layer Memory** — Session history + LLM-compressed long-term memory + LanceDB vector retrieval
- **MCP/Plugin System** — Connect external MCP servers via JSON-RPC with hot-reload support
- **Runtime Tool Creation** — Agent can write, save, and load new Python tools at runtime
- **Self-Repair** — Daily health checks, session diagnostics, error analysis with auto-notification
- **Cron Scheduling** — Persistent one-shot and recurring tasks with timezone awareness
- **Multi-Tenant Router** — Docker-based auto-provisioning with per-user containers
- **Multimodal Support** — Image, video, file, voice, and link handling with ASR
- **Web Search** — Multi-engine search across Tavily, GitHub, and HuggingFace

## Installation

```bash
git clone https://github.com/wangziqi06/724-office.git
cd 724-office
pip install croniter lancedb websocket-client
cp config.example.json config.json
mkdir -p workspace/memory workspace/files
```

Edit `config.json` with your API keys and configuration.

## Basic Usage

```bash
python3 xiaowang.py
```

The agent starts an HTTP server on port 8080. Point your messaging platform webhook to `http://YOUR_SERVER_IP:8080/`.
