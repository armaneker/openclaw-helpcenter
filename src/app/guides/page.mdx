# Guides

Step-by-step guides for common OpenClaw operations.

## Available Guides

### [Create a Slack Agent](/guides/create-slack-agent)
Walk through creating a new specialized OpenClaw agent and connecting it to Slack as an independent bot. Covers Slack app setup, token configuration, routing bindings, and pairing approval.

### [Deployment](/guides/deployment)
Deploy OpenClaw on your own server, configure it as a system daemon, and plan for scaling. Covers recommended server topology, remote access, and phased rollout.
